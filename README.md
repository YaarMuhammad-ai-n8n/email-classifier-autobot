# Email Classifier + Auto-Reply Bot
Built with: n8n + Groq AI + Gmail + Google Sheets

## What It Does
- Reads incoming Gmail emails automatically
- Classifies them as Order or Enquiry using Groq AI
- Generates unique contextual reply for each email
- Sends reply automatically via Gmail
- Logs everything to Google Sheets

## Stack
- n8n (workflow automation)
- Groq LLaMA3 (AI classification + reply generation)
- Gmail API
- Google Sheets API
